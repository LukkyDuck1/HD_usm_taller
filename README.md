# SansaWeigh

Microservicio de estaciones de pesaje de paquetes (Spring Boot).

Documentacion del proyecto (Docsify) publicada en GitHub Pages:
**https://lukkyduck1.github.io/HD_usm_taller/**

Tambien disponible en la carpeta `docs/` (manual de usuario, configuracion del entorno
y arquitectura).

## Ver la documentacion en local

```
npm i -g docsify-cli
docsify serve docs
```

Queda en `http://localhost:3000`.
