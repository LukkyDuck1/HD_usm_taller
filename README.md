# SansaWeigh

Microservicio de estaciones de pesaje de paquetes (Spring Boot).

La documentacion del proyecto esta en la carpeta `docs/` y se publica con **Docsify**
(manual de usuario, configuracion del entorno y arquitectura).

## Ver la documentacion

```
npm i -g docsify-cli
docsify serve docs
```

Disponible en `http://localhost:3000`.
